## 🚀 Sobre mim

Atuo na área de dados desde 2021, trabalhando com:
- Construção e manutenção de pipelines de dados
- Modelagem e arquitetura de dados
- ETL/ELT em ambientes cloud
- Databricks e ecossistema Azure
- Integrações via APIs, DMS e processamento distribuído
- Estruturação de ambientes analíticos para BI e Analytics

Gosto de unir visão analítica com engenharia para criar soluções escaláveis e organizadas.  
Também curto estudar novas tecnologias, automações e projetos paralelos envolvendo dados e IA.
